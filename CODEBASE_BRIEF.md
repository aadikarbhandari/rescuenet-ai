## What each file does (concise)
- `.gitignore` ‚Äî Project config/docs/dependencies.
- `AUDIT_NOTES.md` ‚Äî Project config/docs/dependencies.
- `CODEBASE_BRIEF.md` ‚Äî Project config/docs/dependencies.
- `PHASE2_SUMMARY.md` ‚Äî Project config/docs/dependencies.
- `PRODUCTION_PASS_PLAN.md` ‚Äî Project config/docs/dependencies.
- `README.md` ‚Äî Project config/docs/dependencies.
- `RUNBOOK.md` ‚Äî Project config/docs/dependencies.
- `agents/__init__.py` ‚Äî Python module.
- `agents/coordinator.py` ‚Äî RescueNet AI - Coordinator Agent (classes: CoordinatorAgent)
- `agents/perception.py` ‚Äî Vision + acoustic detection
- `agents/policy_engine.py` ‚Äî RescueNet Policy Engine (classes: PolicyConfig, PolicyEngine)
- `agents/routing.py` ‚Äî Navigation + jamming fallback
- `agents/security.py` ‚Äî Python module. (classes: AlertSeverity, AlertType, SecurityAlert, SecurityAgent; functions: check_for_spoofing, get_security_alerts, log_security_event)
- `agents/state_awareness.py` ‚Äî CRITICAL - live fleet state + decision helpers (classes: StateAwarenessAgent)
- `agents/triage.py` ‚Äî Triage Agent for RescueNet AI (classes: TriageVictim, TriageAgent)
- `agents/voice.py` ‚Äî PersonaPlex integration
- `api/__init__.py` ‚Äî Python module.
- `api/server.py` ‚Äî Python module. (functions: update_state, _api_key_required, _extract_api_key, health, get_status)
- `config/__init__.py` ‚Äî Configuration module for RescueNet AI.
- `config/settings.py` ‚Äî Python module. (classes: RuntimeMode, LogLevel, RescueStation, AirSimSettings, DeepSeekSettings; functions: get_settings, load_settings, get_deepseek_headers, get_llm_client, reset_settings)
- `config.json` ‚Äî Project config/docs/dependencies.
- `config.py` ‚Äî Python module.
- `dashboard/__init__.py` ‚Äî Python module.
- `dashboard/app.py` ‚Äî Python module. (functions: init_system, update_fleet_from_env, create_new_assignments, load_ai_decisions, _build_ops_context)
- `dashboard/dash.py` ‚Äî Python module. (functions: call_glm, main)
- `integration/__init__.py` ‚Äî Integration layer for vendor-specific drone and sensor adapters.
- `integration/adapters.py` ‚Äî Vendor adapter contracts + registry. (classes: NormalizedDrone, NormalizedVictimSignal, DroneAdapter, SensorAdapter, AdapterRegistry)
- `integration/manager.py` ‚Äî Runtime adapter manager. (classes: AdapterSpec, AdapterManager)
- `main.py` ‚Äî RescueNet AI - Main Entry Point (functions: setup_logging, print_startup_banner, print_tick_summary, land_all_drones, normalize_victim_snapshot)
- `requirements.txt` ‚Äî Project config/docs/dependencies.
- `simulation/__init__.py` ‚Äî Python module.
- `simulation/airsim_adapter/__init__.py` ‚Äî AirSim adapter package for RescueNet AI.
- `simulation/airsim_adapter/adapter.py` ‚Äî AirSim adapter for RescueNet AI. (classes: AirSimAdapter; functions: create_airsim_adapter)
- `simulation/airsim_adapter/contracts.py` ‚Äî AirSim data contracts and models for RescueNet AI. (classes: SensorType, SensorStatus, DroneOperationalStatus, DetectionConfidence, MissionCommandType; functions: telemetry_to_drone_state, detection_to_victim_state, create_mission_command, validate_telemetry, validate_detection)
- `simulation/airsim_env.py` ‚Äî Python module. (classes: AirSimEnv)
- `simulation/drone.py` ‚Äî Python module. (classes: Drone)
- `simulation/environment.py` ‚Äî Environment abstraction interface for RescueNet AI. (classes: Environment)
- `simulation/factory.py` ‚Äî Environment factory for creating environment instances based on runtime mode. (classes: SimulationFactory; functions: create_environment, get_environment)
- `simulation/mock_env.py` ‚Äî Deterministic mock disaster environment. (classes: MockDisasterEnv)
- `state/__init__.py` ‚Äî Python module.
- `state/fleet_state.py` ‚Äî Python module. (classes: DroneStatus, MissionStatus, DroneState, VictimState, MissionAssignment)
- `utils/__init__.py` ‚Äî Shared utilities for reliability and operational helpers.
- `utils/observability.py` ‚Äî Operational observability helpers (metrics + structured events). (classes: OpsMetrics; functions: structured_event)
- `utils/reliability.py` ‚Äî Reliability utilities for external service calls. (classes: RetryPolicy, CircuitBreaker; functions: _get_breaker, resilient_post)
